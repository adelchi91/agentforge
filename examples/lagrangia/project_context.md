# Project Context — Lagrangia MecAI

> Generated by project-bootstrap v1.1.0 — Claude target.

## Identity

**Project name:** Lagrangia MecAI

**Description:**
Lagrangia MecAI is a CAD code generation system built for Dessia Technologies. It uses
LangGraph for multi-agent orchestration, RAG pipelines backed by ChromaDB for contextual
knowledge retrieval, and MCP servers to expose mechanical engineering tools to AI agents.
The system generates Python code for mechanical component design, validated against
Dessia's internal CAD framework.

**Stack:** Python, LangGraph, ChromaDB, FastAPI, pytest

## Codebase

Monorepo root contains five top-level packages:

```

code-explorer/      — code analysis and retrieval library (~3 100 lines Python)
```

CI currently runs a single `pytest` sweep across all packages.
