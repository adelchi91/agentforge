## Project Overview

mylib is a Python utility library for data validation. It provides composable, type-safe
validators for common data types — strings, numbers, dates, collections — with a clean
functional API designed to be used as a dependency in data-processing pipelines.

**Stack:** Python 3.11+, pytest, ruff
